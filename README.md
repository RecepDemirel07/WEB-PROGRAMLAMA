# WEB-PROGRAMLAMA
#  Spor Salonu Yönetim ve Randevu Sistemi

Bu proje, **Sakarya Üniversitesi Web Programlama Dersi** dönem ödevi kapsamında geliştirilmiş; spor salonları, antrenörler ve üyeler arasındaki etkileşimi dijitalleştiren web tabanlı bir yönetim sistemidir.

## Proje Hakkında
Sistem, spor salonu yöneticilerinin salonlarını, hizmetlerini ve antrenörlerini kolayca yönetmesini sağlarken; üyelerin antrenörlerin uzmanlık alanlarına göre uygun saatlerde randevu alabilmesine olanak tanır. Proje, **ASP.NET Core MVC** mimarisi üzerine inşa edilmiş olup **Entity Framework Core** (Code First) yaklaşımı kullanılarak geliştirilmiştir.

## Özellikler
* **Kullanıcı Yönetimi:** Admin ve Üye olmak üzere iki farklı rol yapısı (ASP.NET Core Identity).
* **Salon Yönetimi:** Spor salonu ekleme, güncelleme, silme ve listeleme işlemleri.
* **Antrenör & Hizmetler:** Antrenörlerin uzmanlık alanlarına göre tanımlanması ve hizmetlerle eşleştirilmesi.
* **Randevu Sistemi:** Üyelerin antrenörlerin müsaitlik durumuna göre randevu talep etmesi ve yönetimi.
* **REST API:** Verilerin dışarıya sunulması için API desteği.
* **Yapay Zeka Desteği:** (Opsiyonel) Kullanıcı verilerine göre basit egzersiz/diyet öneri sistemi.

## Kullanılan Teknolojiler
* **Dil:** C#
* **Framework:** ASP.NET Core 8.0 MVC
* **Veritabanı:** MS SQL Server (LocalDB)
* **ORM:** Entity Framework Core
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Versiyon Kontrol:** Git & GitHub
